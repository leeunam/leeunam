  > Developed a predictive analytics solution for fleet maintenance using historical operational data. I was responsible for data normalization and for implementing **Linear Regression** and **KNN regression** models to estimate maintenance intervals and support preventive planning.

- [Ctrl + S — Building Inspection Management Platform](https://github.com/Inteli-College/2025-1B-T16-IN02-G03)  
  > Developed a web platform to support and standardize building inspections in collaboration with **IPT**. I worked on the **prototype**, **front-end with HTML/CSS**, **backend logic in JavaScript**, and **database modeling with Supabase/PostgreSQL**.

  > Web3 education platform created to make blockchain learning more accessible and connect qualified talent with companies. Built during a hackathon, the project won **1st place in the Arbitrum track** and ranked **3rd overall**.
